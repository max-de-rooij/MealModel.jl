# MealModel
The Julia version of the Mixed Meal Model. Originally developed by S.D. O'Donovan[^1]

## Getting Started
1. To get started, first clone the github repository to your own device
2. Then, open the notebook "MealModel.ipynb" in your Jupyter Notebook viewer, and run the first cell to set-up the environment.
